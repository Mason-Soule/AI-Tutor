# AI-Tutor
AI-powered tutor that helps students prepare for exams by generating practice problems and explaining key concepts through guided lessons.

Potential Languages & Tools

Frontend: React (with Vite) Pair it with Tailwind CSS for styling
Backend: Node.js with Express (or Python with FastAPI)
PDF handling: On the frontend, pdf.js to render PDFs. On the backend, use PyMuPDF or pdfplumber (Python) or pdf-parse / pdfjs-dist (Node.js) to extract text to send to Claude.
Database: PostgreSQL for storing user sessions, exam history, and scores
